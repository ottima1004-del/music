# Music

A basic `Next.js` + `Vercel` app.

## Production

Production URL: `https://music-sandy-eta.vercel.app/`

## Getting Started

1. Install Node.js 20 or newer.
2. Install dependencies.
3. Start the development server.

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Deploy To Vercel

```bash
npm install -g vercel
vercel
```

You can also push this project to Git and import it from Vercel.
